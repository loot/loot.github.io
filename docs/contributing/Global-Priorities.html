--- layout: page title: Global Priorities ---

Global priorities are used to sort plugins on a "global" scale. LOOT will normally only sort plugins that conflict with each other, but setting a global priority will allow some plugins to load earlier or later than others, even if they don't conflict. This can be useful to make sure that e.g. unofficial patch plugins are loaded earlier than all other non-official plugins. Details about how they work in practice can be seen in the Metadata Syntax documentation: see the description for the "global_priority" field under "Plugin Data Structure".

Below are the global priorities as they are grouped for the games currently supported by LOOT. In practice, there are existing entries that use different priority values, they were set before values were standardised, and are maintained for compatibility.
